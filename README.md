# ISQUIHUB  
**Plataforma web interna para la gestión administrativa y de Recursos Humanos en ISQUISA**

## Descripción general

**ISQUIHUB** es una plataforma web interna desarrollada como proyecto de **Residencias Profesionales** en **ISQUISA S.A. de C.V.**, cuyo objetivo principal es la **digitalización, centralización y optimización de los procesos administrativos y de Recursos Humanos**, tanto a nivel **corporativo** como en **plantas operativas**.

El proyecto nace a partir del análisis directo de la operación real de la empresa, identificando problemáticas como:
- Uso excesivo de formatos físicos
- Procesos manuales no estandarizados
- Duplicidad de información
- Falta de trazabilidad y control histórico
- Dependencia de múltiples plataformas no integradas

ISQUIHUB se plantea como una **solución tecnológica escalable**, alineada a las necesidades actuales de ISQUISA y con visión a futuro para soportar el crecimiento de la organización.

## Enfoque del proyecto

El desarrollo de ISQUIHUB tiene un **enfoque técnico y de ingeniería en sistemas computacionales**, integrando:
- Análisis de procesos
- Diseño de arquitectura de software
- Modelado de bases de datos
- Desarrollo web modular
- Automatización de flujos administrativos
- Integración con plataformas existentes (como WorkBeat y sistemas de control de accesos)

El proyecto no se limita a la programación, sino que abarca el **levantamiento de requerimientos**, la **documentación técnica**, la **estructura organizacional** y la **operación real de Recursos Humanos**.

## Rol del desarrollador (Residencias Profesionales)

Como **Residente de Recursos Humanos con perfil de Ingeniero en Sistemas Computacionales**, mis responsabilidades dentro del proyecto incluyen:

- Análisis de procesos administrativos y de RH en corporativo y plantas
- Diseño funcional y técnico de la plataforma ISQUIHUB
- Definición de módulos, flujos y reglas de negocio
- Diseño de la estructura de base de datos
- Desarrollo de interfaces web (HTML, CSS, JavaScript)
- Implementación de lógica de negocio del lado del cliente
- Documentación técnica y funcional del sistema
- Apoyo en la gestión de plataformas de RH (WorkBeat)
- Estructuración de puestos, organigramas y catálogos organizacionales
- Propuesta de mejoras tecnológicas alineadas a la operación real

## Alcance funcional de ISQUIHUB

La plataforma contempla, entre otros, los siguientes módulos:

### Recursos Humanos
- Gestión de colaboradores
- Control de incidencias y asistencias
- Administración de puestos y organigramas
- Gestión de vacaciones y permisos
- Control de documentación del personal
- Catálogos organizacionales (departamentos, centros de costo, regiones)

### Administración y Operación
- Gestión de requisiciones internas
- Control de inventarios específicos (ej. bodega de uniformes)
- Registro de movimientos administrativos
- Flujos de validación y autorización

### Estructura corporativa y plantas
- Separación lógica entre corporativo y plantas
- Configuración por región, sucursal y centro de trabajo
- Roles y permisos por tipo de usuario
- Adaptación a la operación real de cada planta
